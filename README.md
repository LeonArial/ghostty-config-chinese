Ghostty Config 是一个为 [Ghostty Terminal](https://github.com/ghostty-org) 打造的精美配置生成器。功能包括轻松修改设置、字体演练场，以及用于光标、选区甚至调色板的大量预览和演示。更多功能即将推出，包括快捷键验证。

<p align="center">
  <img src="./src/lib/images/截图.png" alt="GUI Preview">
</p>

这是一个 Web 工具，旨在帮助用户更方便地配置 Ghostty 终端。

### 原项目信息

* **原项目仓库**: [zerebos/ghostty-config](https://github.com/zerebos/ghostty-config)
* **在线演示 (英文原版)**: [ghostty.zerebos.com](https://ghostty.zerebos.com/)

### 本地运行

1. 克隆本仓库
   ```bash
   git clone https://github.com/YOUR_USERNAME/ghostty-config.git
   cd ghostty-config
   ```
2. 安装依赖
   ```bash
   pnpm install
   ```
3. 启动开发服务器
   ```bash
   pnpm dev
   ```

由于本工具旨在与桌面版 Ghostty 终端配合使用，因此在移动设备上的体验不佳。
